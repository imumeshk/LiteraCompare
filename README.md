# LiteraCompare

This repository provides tools and utilities to manage and utilize the Litera Compare Desktop application in a more efficient way.

## Tools Included

- **Litera Compare Tool**: A comprehensive management utility for Litera Compare to handle user file cleanup, backups, system requirements validation, installations, license information, custom settings, and Office add-ins.
- **Litera Compare Bulk Compare Utility**: A bulk comparison tool allowing users to automatically compare multiple files or directories against each other using various Litera comparison engines.
