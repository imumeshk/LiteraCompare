# Litera Compare Tool

The Litera Compare Tool is a management utility designed to streamline the administration and troubleshooting of the Litera Compare Desktop application. It provides a user-friendly graphical interface (built with PowerShell and WPF) that consolidates various administrative and maintenance tasks into a single dashboard.

## How it is Useful

Administrators and power users often have to navigate through multiple system directories, registry keys, and configurations to manage Litera Compare. This tool solves that problem by bringing everything into one place. It is highly useful for:
- **Quick Troubleshooting**: Diagnose issues by easily checking system requirements, validating prerequisites, and viewing Litera logs in real-time.
- **Maintenance**: Perform routine maintenance like cleaning up temporary files and caches that might cause the application to hang or crash.
- **Configuration Management**: Backup and restore Litera Compare settings seamlessly when migrating users or resetting environments.
- **Add-in Management**: Quickly enable, disable, or fix Litera Compare Office add-ins (Word, Excel, PowerPoint) without digging into Microsoft Office options.

## Key Features

- **User File Cleanup**: Easily clear out temporary files and leftover data.
- **Backup & Restore**: Safely back up and restore Litera Compare configurations.
- **System Requirements/Pre-requisites**: Quickly verify if the system meets the prerequisites to run Litera Compare.
- **Install/Uninstall**: Manage the installation and uninstallation of the software.
- **License Info**: Check current license details and status.
- **Compare Customizations**: Manage custom settings for document comparison.
- **Office Add-in Management**: Manage Litera integrations with Microsoft Office applications.
- **Log Viewer**: Access and monitor Litera Compare logs to diagnose issues quickly.
